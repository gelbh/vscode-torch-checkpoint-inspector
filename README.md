# Torch checkpoint inspector

[![VS Marketplace](https://img.shields.io/badge/VS%20Marketplace-listing-007ACC?logo=visualstudiocode)](https://marketplace.visualstudio.com/items?itemName=gelbhart.vscode-torch-checkpoint-inspector)
[![Open VSX](https://img.shields.io/badge/Open%20VSX-listing-573798?logo=eclipseide)](https://open-vsx.org/extension/gelbhart/vscode-torch-checkpoint-inspector)
[![License](https://img.shields.io/github/license/gelbh/vscode-torch-checkpoint-inspector)](https://github.com/gelbh/vscode-torch-checkpoint-inspector/blob/main/LICENSE)
[![VS Code](https://img.shields.io/badge/VS%20Code-%5E1.85.0-purple)](https://code.visualstudio.com/)

Inspect `.pt` / `.pth` in a custom editor. Needs Python with **PyTorch**; **pandas** / **numpy** if checkpoints embed DataFrames. Interpreter: `torchCheckpointInspector.pythonPath`, else Python extension, else `python3`. Optional `torchCheckpointInspector.allowUnsafeLoad` (trusted files only).

**Install:** [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=gelbhart.vscode-torch-checkpoint-inspector) · [Open VSX](https://open-vsx.org/extension/gelbhart/vscode-torch-checkpoint-inspector) (Cursor / VS Code–OSS).
